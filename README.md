# Student Exam Pass Prediction System

## Introduction

The Student Exam Pass Prediction System is a Machine Learning project that predicts whether a student is likely to Pass or Fail based on academic, demographic, and behavioral factors.
The project uses Logistic Regression, Exploratory Data Analysis (EDA), and data visualization techniques to identify key factors influencing student performance and support data-driven educational decisions.

## Objectives

* Analyze student performance data.
* Identify factors affecting exam success.
* Predict Pass/Fail outcomes using Logistic Regression.
* Detect at-risk students early.
* Generate insights through visualizations.
* Support academic planning and intervention strategies.

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn
* Jupyter Notebook

## Data Analysis Process

**Data Cleaning**

* Missing value handling
* Duplicate removal
* Data type verification
* Categorical data encoding

**Exploratory Data Analysis**

* Student performance analysis
* Attendance analysis
* Study habits analysis
* Pass vs Fail comparison
* Correlation analysis

**Machine Learning**

* Logistic Regression Model
* Train-Test Split
* Prediction of Pass/Fail outcomes
